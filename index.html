Google Sheets Connection

  1. Open Google Sheets → new blank spreadsheet
  2. Click Extensions → Apps Script
  3. Delete all code, paste in the google-apps-script.js file
  4. Click Save → Deploy → New Deployment
  5. ⚙ gear → Web App → Execute as: Me → Who has access: Anyone
  6. Click Deploy → authorise → copy the URL
